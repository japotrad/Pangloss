Package tmf

The tmf package has been developed based on ISO 16642 and has two main responsabilities :
  1. Represents the terminological data of a Freeplane map as a Terminological Markup Framework datamodel (methods populate); The root of the termbase is the tdc object;
  2. Outputs this termbase into an XML file (methods toGMT;)
