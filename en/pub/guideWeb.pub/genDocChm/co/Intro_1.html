Input an Entry

Here is the guideline to fit a glossary Entry into a mind map node:

  1. A style is applied to a map node, so that it becomes an glossary Entry. The name of the style is: Entry.

  2. The most significant features are stored in the node core.

  3. Most Entry and Term features can be stored either in the node note, details or attributes.

Apply the Entry style to the node

  1. Select the node.

  2. In the style drop-down box, select

Compléments

The Entry style has to be defined beforehand. To define a style, refer to Freeplane wiki.

Input the most significant features in the node core

  1. Double-click the node core.

  2. Input the term describing the best the concept you are talking about.

  3. Optionally, if you work mainly with two languages, append an equal = followed by the appropriate term in the other language.

Compléments
Conceptual vs. Bilingual Entries

If a node core contains no equal character, it is considered as a Conceptual Entry. Such Entry focuses on the concept itself (but may be multilingual). The language to be input in the node core should correspond to the languageID feature.

If a node core contains an equal character, it is considered as a Bilingual Entry. Such Entry focuses on the term in a primary language called the source language. The language to be input in the node core should correspond to the sourceLanguage feature.

Conceptual Entry

If you study economics, you are likely to create an gross national product conceptual entry.

If you have to translate into spanish the term "gross national product", you should create the following bilingual entry: Gross national product=Producto Nacional Bruto

The most important thing to remember if you want to use Pangloss is to apply the Entry style to all nodes containing glossary entries. If this style is not defined in your map, just create it yourself.
